# sudoku-csp-solver-backtracking-ac3
Constraint Satisfaction Problem (CSP)-based Sudoku solver implementing backtracking search, forward checking, and AC-3 algorithm. Efficiently solves puzzles of varying difficulty while tracking search performance through backtrack calls and failures. Designed for AI coursework and experimentation.
# Sudoku CSP Solver

This project implements a Sudoku solver using:

- Backtracking Search
- Forward Checking
- AC-3 Algorithm

## Input Format
- 9 lines per file
- Each line has 9 digits (0 = empty)

## Features
- Solves Sudoku of varying difficulty
- Counts:
  - Backtrack calls
  - Backtrack failures

## How to Run
python solver.py
